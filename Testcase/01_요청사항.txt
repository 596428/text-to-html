=====================================================
견적서 관리 화면 제작 요청사항
=====================================================

## 📋 박스 1: 상단 탭 네비게이션
- 위치: 최상단, 전체 너비
- 내용: 4개의 탭 버튼 (견적서 관리 / 계약서 관리 / 해약서 관리 / 신청내역)
- 디자인:
  * 첫 번째 탭 "견적서 관리"는 활성 상태 (검은색 테두리, 흰색 배경)
  * 나머지 탭은 비활성 상태 (회색 텍스트)
  * 탭들이 가로로 나열되고 하단에 구분선

## 📋 박스 2: 검색 필터 영역 (좌측)
- 위치: 탭 아래, 왼쪽 절반
- 내용:
  * "지정현황" 라벨 (연한 파란색 배경)
  * "사업장수" 입력 필드
  * "근로자수" 입력 필드

## 📋 박스 3: 검색 필터 영역 (우측)
- 위치: 탭 아래, 오른쪽 절반
- 내용:
  * "사업현황" 라벨 (연한 파란색 배경)
  * "사업장수" 입력 필드
  * "근로자수" 입력 필드

## 📋 박스 4: 검색 내용 입력 영역
- 위치: 검색 필터 아래, 전체 너비
- 내용:
  * "검색내용" 라벨 (연한 파란색 배경)
  * "보건관리" 드롭다운 (오른쪽 끝)
  * "등록" 버튼 (파란색, 오른쪽 끝)

## 📋 박스 5: 검색 조건 테이블
- 위치: 검색 내용 아래
- 내용: 3행 × 4열 테이블
  * 행 1: 사업장명 | 신성기업(주) 드롭다운 + 조회/추가 버튼 | 비교견적 여부 | 비교견적 체크박스
  * 행 2: 주소 | (빈칸) | 전화번호 | 팩스번호
  * 행 3: 이메일 | (빈칸) | 담당자/직책 | 휴대폰
- 디자인:
  * 좌측 라벨 컬럼은 연한 파란색 배경
  * "조회" 버튼은 파란색
  * "추가" 버튼은 흰색 배경 + 검은색 테두리

## 📋 박스 6: 금액 입력 영역
- 위치: 검색 조건 테이블 아래, 전체 너비
- 내용: 1행 × 8열 테이블
  * 열: 인당단가 | (입력필드) | 인원 | (입력필드) | 월급액 | (입력필드) | 개월 | (입력필드) | 연간급액 | (입력필드)
- 디자인: 라벨 셀은 연한 파란색 배경

## 📋 박스 7: 비고 입력 영역
- 위치: 금액 입력 아래
- 내용:
  * "비고" 라벨 (연한 파란색 배경, 좌측)
  * 넓은 텍스트 입력 영역 (우측)

## 📋 박스 8: 업무분야 영역
- 위치: 비고 아래
- 내용:
  * "업무분야" 버튼 (파란색 배경, 흰색 텍스트, 좌측 하단)
  * 넓은 텍스트 입력 영역 (우측)

## 📋 박스 9: 견적서 발행 내역 테이블 헤더 + 버튼
- 위치: 업무분야 아래
- 내용:
  * "견적서 발행 내역" 제목 (연한 파란색 배경, 좌측)
  * 우측에 4개 버튼: "당월전체" / "당해연도" / "총월미승인" / "총해미승인" (모두 파란색)

## 📋 박스 10: 견적서 발행 내역 테이블
- **팝업 기능 있음**
- 팝업 버튼 텍스트: "조회" (테이블 내 파란색 버튼)
- 위치: 견적서 발행 내역 헤더 아래
- 내용: 데이터 테이블
  * 헤더: 견적서 번호 | 사업장명 | 견적구분 | 계약서 | 출력 | 매일전송 | 문자전송 | 인원 | 인당단가 | 총금액 | 비교견적 | 삭제
  * 각 헤더 아래 "검색:" 드롭다운 3개
  * 데이터 행 1개:
    - E202500008 | 지멘스 | 직무스트레스평가 | 계약서(파란색) | 출력(파란색) | 매일전송(파란색) | 문자전송(파란색) | 100 | 9,100 | 1,421,300 | 체크박스 | 삭제(파란색)
- 디자인:
  * 헤더는 파란색 배경, 흰색 텍스트
  * 파란색 버튼들이 많음 (조회, 출력, 매일전송, 문자전송, 삭제)
  * 금액은 오른쪽 정렬

- "사업장명:" 라벨 + "신성기업(주)" 입력 필드
- "검색" 버튼 (파란색)
- "26299" 숫자 표시
- "조회" 버튼 (파란색)

#### 섹션 2: 검색 결과 테이블
테이블 헤더:
- 사업장 코드 | 사업장명 | 사업장 등록번호 | 대표자 | 주소 | 검색여부 | 검색항목

데이터 행:
- A10240 | 신성기업(주) | 608-82-04034 | 임현태 | 경남 창원시 마산회원구 자유무역2길 38 | Y | 직무(부산 건강진단_정체이)

디자인:
- 테이블 헤더는 회색 배경
- 데이터 행은 흰색 배경
- 테두리는 회색 실선

#### 섹션 3: 하단 버튼 영역
(팝업창에는 확인/취소 버튼이 보이지 않지만, 일반적으로 닫기 버튼은 우측 상단에 위치)
